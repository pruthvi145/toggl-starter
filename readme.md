# Toggl Starter Utility

## Requirements
 * zenity
 * toggl-cli
 * notify-send

## Installation

### Install gnome utilities: zenity and notify-send

#### ubuntu
```bash 
sudo apt-get install zenity
sudo apt-get install libnotify-bin
```

### Install toggl-cli

```bash
pip install togglCli
```

## Run

```bash
python3 start.py
```
